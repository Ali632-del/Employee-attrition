# شرح مشروع Employee Attrition Prediction System بالعربية

## 📌 المشروع عبارة عن إيه؟

المشروع هو **تطبيق ويب تفاعلي** يتنبأ بـ **احتمالية ترك الموظفين للشركة** باستخدام **الذكاء الاصطناعي والتعلم الآلي**.

### الهدف:
- تحليل بيانات الموظفين
- التنبؤ بمن قد يترك الشركة
- مساعدة الشركة على الاحتفاظ بالموظفين

---

## 📂 الملفات الموجودة في المشروع

### 1️⃣ **app.py** (24 KB) - التطبيق الرئيسي
**ده إيه؟**
- هو الملف الرئيسي للتطبيق
- يحتوي على واجهة المستخدم التفاعلية
- مكتوب بـ **Streamlit** (مكتبة Python لعمل تطبيقات ويب)

**فيه إيه بداخله؟**
- صفحة Home: معلومات عن المشروع
- صفحة Data Overview: عرض البيانات
- صفحة Data Analysis: رسوم بيانية وتحليلات
- صفحة Predictions: إدخال بيانات موظف والتنبؤ
- صفحة Model Info: معلومات عن النموذج

---

### 2️⃣ **analysis_notebook.py** (13 KB) - تحليل البيانات
**ده إيه؟**
- ملف Python يحتوي على كل عملية التحليل
- يقرأ البيانات ويحللها
- يدرب النموذج (الذكاء الاصطناعي)

**فيه إيه بداخله؟**
```python
1. تحميل البيانات من CSV
2. تنظيف البيانات (حذف الأخطاء)
3. تحليل البيانات (رسوم بيانية)
4. تدريب 3 نماذج مختلفة
5. اختيار أفضل نموذج
6. حفظ النموذج في ملف
```

---

### 3️⃣ **best_model.pkl** (139 KB) - النموذج المدرب
**ده إيه؟**
- هو النموذج الذي تم تدريبه بالفعل
- يحتوي على الذكاء الاصطناعي الذي يتنبأ
- ملف ثنائي (binary file) - لا تفتحه بـ notepad

**مثال:**
```
إدخال: بيانات موظف (العمر، الراتب، الساعات الإضافية)
↓
النموذج يعالج البيانات
↓
إخراج: احتمالية ترك الموظف (80% مثلاً)
```

---

### 4️⃣ **preprocessor.pkl** (6.8 KB) - معالج البيانات
**ده إيه؟**
- يحول البيانات الخام إلى صيغة يفهمها النموذج
- مثل مترجم يحول اللغة

**مثال:**
```
إدخال: "الراتب = 5000 دولار"
↓
المعالج يحوله
↓
إخراج: "0.75" (رقم يفهمه النموذج)
```

---

### 5️⃣ **emp_attrition.csv** (223 KB) - البيانات الأصلية
**ده إيه؟**
- ملف Excel بصيغة CSV (جدول)
- يحتوي على بيانات 1,470 موظف
- فيه 35 عمود (معلومات عن كل موظف)

**الأعمدة الموجودة:**
```
- العمر (Age)
- الراتب الشهري (Monthly Income)
- الساعات الإضافية (Overtime)
- سنوات العمل (Years at Company)
- الرضا الوظيفي (Job Satisfaction)
- وغيرها...
```

## ⚠️ مشاكل شائعة والحل

### المشكلة 1: "ModuleNotFoundError"

**السبب:** لم تثبت المكتبات

**الحل:**
```bash
pip install -r requirements.txt
```

---

### المشكلة 2: "streamlit: command not found"

**السبب:** لم تفعل البيئة الافتراضية

**الحل:**
```bash
# على Windows:
venv\Scripts\activate

# على Mac/Linux:
source venv/bin/activate
```

---

### المشكلة 3: "FileNotFoundError: emp_attrition.csv"

**السبب:** ملف البيانات ليس في نفس المجلد

**الحل:**
- تأكد أن `emp_attrition.csv` في نفس مجلد `app.py`

---

### المشكلة 4: "Port 8501 is already in use"

**السبب:** التطبيق يعمل بالفعل

**الحل:**
```bash
# استخدم port مختلف
streamlit run app.py --server.port 8502
```
